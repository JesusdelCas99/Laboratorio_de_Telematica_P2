# Laboratorio_de_Telematica_P2

Esta aplicación GUI desarrollada en la práctica 2 de Laboratorio de Telemática, permite
la resolución del ejercicio teórico estudiado en la parte de teoría de dicha asignatura.

La interfaz permite la introducción de distintos datos para los que calculará todos los
parámetros necesarios para el despliegue de red. Adicionalmente muestra un breve esquema
de esta.

Adicionalmente incorpora un archivo .db, a modo de base de datos para los distintos códecs
que se pudieran implementar.
