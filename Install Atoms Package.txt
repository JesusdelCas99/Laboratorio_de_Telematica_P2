Solución al problema de "MAXIMUM STACK SIZE EXCEED" que no nos permite descargar paquetes

1º Instalamos Node.js: https://nodejs.org/en/

2º Nos vamos a las variables de entorno y en variables de usuario y variables de sistema chequeamos:
   Variables de usuario:
   C:\Users\Jesus\AppData\Local\atom\bin
   C:\Users\Jesus\AppData\Roaming\npm
   C:\Program Files\nodejs

3º Variables del sistema:
   %USERPROFILE%\AppData\Local\atom\bin
   C:\Program Files\nodejs\

4º Cerramos la terminal y la reabrimos y escribimos el siguiente comando: npm rebuild

5º Ya podremos instalar los paquetes desde la terminal usando el siguiente comando: 'apm install Package'.
   Por ejemplo, para instalar el paquete "script": apm install script
